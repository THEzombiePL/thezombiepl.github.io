Anti‑VPN API to lekkie i wydajne REST API służące do wykrywania,

czy dany adres IP należy do VPN‑a, proxy lub infrastruktury data center.


API bazuje na automatycznie aktualizowanych listach CIDR

i zostało zaprojektowane z myślą o łatwej integracji z aplikacjami webowymi,

backendami oraz systemami security.


Projekt udostępnia publiczną dokumentację w Swaggerze

oraz wspiera obsługę nagłówków proxy (X‑Forwarded‑For),

co pozwala na poprawne działanie za reverse proxy lub load balancerem.
