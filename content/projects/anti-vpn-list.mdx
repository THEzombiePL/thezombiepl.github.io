Anti‑VPN‑List to projekt generujący aktualne listy adresów IP i zakresów CIDR
należących do VPN‑ów, proxy oraz data center.

Lista jest automatycznie budowana na podstawie wielu publicznych źródeł
i regularnie aktualizowana przy użyciu GitHub Actions.

Projekt został zaprojektowany z myślą o zastosowaniach security - takich jak
ochrona API, filtrowanie ruchu, anty‑fraud czy ograniczanie dostępu z sieci
anonimizujących.
